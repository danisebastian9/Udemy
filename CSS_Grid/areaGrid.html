Using grid-template-areas: "header header" "content sidebar" "footer footer"; The content can be named and placed according to a structured way.
The implicit grill are the lines added after the values that were declared, in this case until footer. The rest of content is added to the implicit grill and it can be controlled with grid-auto-rows and grid-auto-flow.
